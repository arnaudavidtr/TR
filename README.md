# Alerta: Phishing — Projecte TR

## Descripció
Pàgina educativa per aprendre a detectar intents de phishing. Inclou:

- Simulador d’emails sospitosos
- Quiz autocorrectiu
- Analitzador d’URL (sintàctic)
- Checklist imprimible
- Comptador de visites (CountAPI)

**Privadesa:** No es recullen contrasenyes ni dades personals. El comptador només mostra un número agregat públic.

## Objectiu
L’objectiu és conscienciar sobre la seguretat online i ensenyar bones pràctiques davant correus sospitosos o pàgines fraudulentas.

## Com utilitzar
1. Obre `index.html` o visita via GitHub Pages.
2. Explora els exemples de phishing i legítims.
3. Fes el quiz autocorrectiu per comprovar el teu coneixement.
4. Utilitza l’analitzador d’URL per practicar la detecció de senyals sintàctics.
5. Imprimeix el checklist per tenir-lo a mà.

## Tecnologies
- HTML5 / CSS3 / JavaScript
- CountAPI per comptador públic de visites
- Totalment client-side, sense back-end ni registre de dades
